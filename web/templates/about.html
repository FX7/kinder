Help

Providers:

Select the providers you want to get movie suggestions from.

Genres:

Anti-Genres are genres that should be ignored. A movie with a selected Anti-Genre will not be presented for voting.
Must-Genres are genres that should be included. A movie without a selected Must-Genre will not be presented for voting.
If no genre infos could be fetched for a movie, it will be presented anyway.

Miscellaneous:

The Max age defines the maximum age (FSK) a movie should be rated for to be presented.
The Max dur. defines the maximum duration a movie should have to be be presented.
The Min release year movies released before this year, will be ignored.
The Max release year movies released after this year, will be ignored.
If no information about age, duration, or release year is found for a movie, it will be presented anyway.

Overlays:

Defines overlays, that are put over the movie poster. Most of them are just informational. But the trailer play button also gives you the ability to watch trailer of the movie to vote.

Limits:

Limits define when a session should end.
You can define a time limit in minutes. The session will end, after the given minutes after creation.
You can define a vote limit. The session will end, after the given amoung of votes (= movies to vote for). This is a "personal" limit. So the session might go on for someone who has not reached the vote limit yet, while it is already over for someone who already reached the vote limit.
You can define a match limit. The session will end, as soon as the amoung of perfect matches is reached.
